# Reading lines of file:
- Method 1 : for line in f:        - low memory usage
- Method 2 : f.readline()          - low memory usage   - read one line
- Method 3 : f.readlines()         - high memory usage  - reads entire file
- both 1 & 2 are similar 

- If we have a text file, basically a long string with \n characters
- we can iterate over each line by doing: for line in f
